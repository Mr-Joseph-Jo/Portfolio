About me

Hello! I’m Joseph Jo, a B.Tech student in Information Technology at Rajagiri School of Engineering and Technology. My journey into tech started with a curiosity about how websites work, which soon grew into a strong passion for building and deploying full-stack web applications.

I specialize in web development with a focus on self-hosted solutions. I’m currently working on EduTrackr, a Flask-based web app backed by MySQL, and constantly exploring tools like Cloudflare Tunnels, Docker, and VS Code Web Live to improve my workflow. My frontend stack includes HTML, CSS, and JavaScript, and I use tools like Metabase and Power BI for data visualization.

During my internship at Tamimah Digital India Pvt. Ltd., I developed an OCR mobile app using Flutter and Dart, gaining hands-on experience with REST APIs, SQL Server, and GitHub. I’ve also explored fields like Soft Computing and IoT, and I enjoy solving real-world problems through smart, scalable tech.

My goal is to become a well-rounded full-stack developer, confident in both development and deployment, always eager to learn, build, and create meaningful solutions.
